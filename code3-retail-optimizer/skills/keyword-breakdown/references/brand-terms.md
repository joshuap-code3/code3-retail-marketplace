# Brand Term Reference

Use these lists for automatic brand vs. non-brand classification. Update this file as
new clients or brands are added. Classification should also catch misspellings,
abbreviations, and brand + modifier combinations beyond the literal lists below.

zevia tonic, zevia ginger beer, zevia lemon lime twist, zevia grape, zevia orange,

## Canon — Camera BU

Brand terms include: canon, canon camera, canon eos, canon rebel, canon r5, canon r6,
canon r7, canon r8, canon r10, canon r50, canon r100, canon 5d, canon 6d, canon 7d,
canon 90d, canon lens, canon ef, canon rf, canon speedlite, canon mirrorless,

**Note on Canon:** Canon Printer and Canon Camera are separate business units.
If a dataset contains both, segment them separately in every analysis table. Never
blend Canon Printer and Canon Camera metrics.
